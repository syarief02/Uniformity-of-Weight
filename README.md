# Uniformity of Weight Test — Comprehensive Guide

> ⚖️ **Ujian Keseragaman Berat** — NPRA SOP PKKK/300/UAT/037

Interactive web-based comprehensive guide for the Uniformity of Weight Test based on NPRA's official Standard Operating Procedure. Covers procedures for uncoated tablets, coated tablets, hard capsules, soft capsules, and lozenges per USP 43 <2091>.

## 🔗 Live

**[syariefazman.com/uow](https://syariefazman.com/uow)**

## 📋 Contents

- **Purpose & Scope** — Traditional pharmaceutical product testing
- **Responsibilities** — Ketua Seksyen, Ketua Unit, Pegawai Farmasi, Penolong Pegawai
- **Procedure** — Step-by-step for each dosage form with expandable sections
- **USP 43 Acceptance Limits** — Weight variation tables and formulas
- **Decision Flow** — Pass/Fail criteria summary
- **Quality Records** — PKKK/UAT/002A, 002B, 002C
- **Revision History** — Full document history

- HTML5 / CSS3 / JavaScript
- Glassmorphism dark/light theme
- Responsive design
- Scroll reveal animations
